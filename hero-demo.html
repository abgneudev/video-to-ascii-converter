Usage

This demo shows a procedurally generated animation. For production use, preprocess your video:

1. Install the preprocessor

cd preprocessor
npm install
npm run build

2. Process your video

npx ascii-process \
  --input hero-video.mp4 \
  --output public/ascii-data/hero.bin \
  --cols 120 \
  --fps 24 \
  --color-mode mono

3. Use in your site

import { ASCIIHero } from './components/ASCIIHero';
import './components/ASCIIHero/ASCIIHero.css';

const hero = new ASCIIHero('#hero', {
  src: '/ascii-data/hero.bin',
  fadeIn: true,
  speed: 1
});

// Set overlay content
hero.setOverlay(`
  <h1>Your Brand</h1>
  <p>Tagline here</p>
`);

File Size Comparison

For a 10-second loop at 24fps:

Original MP4:    ~500KB - 2MB
ASCII Binary:    ~50-150KB (gzipped)
Runtime CPU:     <5% (vs 30-50% for real-time)
